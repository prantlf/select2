Native

The native look&smp;feel is enabled by the original Select2 interface, although the select2.css just imports select2-base.css and select2-native.css now.

  <link rel="stylesheet" href="select2.css">
  <script src="select2.js"></script>
  $('...').select2();

jQuery UI

The jQuery UI mode expects loading the select2-base.css only instead of the select2.css and passing the jqueryu-ui theme mode to the initializer.

  <link rel="stylesheet" href="select2-base.css">
  <link rel="stylesheet" href="select2-jquery-ui.css">
  <script src="select2.js"></script>
  <script src="select2-jquery-ui.js"></script>
  $('...').select2({ theme: 'jquery-ui' });

The jQuery UI mode is not used by default, because the jQuery UI is often used only partially with other GUI framework. If full jQuery UI including the themes is used on the page, it can be set up to be used by default to save the constructor parameter.

  $.fn.select2.defaults.theme_default = 'jquery-ui';
