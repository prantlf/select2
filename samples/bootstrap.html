Bootstrap

The Bootstrap mode depends on the full select2.css which includes the native styling. It is pre-selected as the default theme after including select2-bootstrap.js, because Bootstrap cannot be easily combined with other GUI frameworks on a single page and once included, it is used completely.

  <link rel="stylesheet" href="select2.css">
  <link rel="stylesheet" href="select2-bootstrap.css">
  <script src="select2.js"></script>
  <script src="select2-bootstrap.js"></script>
  $('...').select2();
